Apartment Parking Allocation System

A simple yet practical web based parking allocation system designed to solve one of the most common apartment management problems, fair and hassle free parking slot allocation This project helps apartment associations or building managers allocate parking slots efficiently, avoid duplicate assignments, and maintain proper records using Excel export function.

In scenarios like: 
A) A resident returns home after work only to find someone parked in their assigned slot. 
B) During initial apartment meetings no one knows who was assigned which parking space.
C) A new tenant arrives but the apartment association has no proper parking allocation record.

This project helps solving these real world day to day issues.

Technologies Used

1.Frontend HTML CSS
2.Backend Python Flask
3.Libraries Used OpenPyXL (Excel file handling)
